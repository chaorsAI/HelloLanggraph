# LangGraph 基础

- helloWorld.py：第一个 LangGraph 程序，最简单的流程使用

- multiple_schema.py：多 Schema 限制输出字段

- multiple_schema_private.py：多 Schema 实现“特定节点（a、b）间传递私有字段（如ab_private），其他节点（c）无法访问”

- reducers.py：合并规则引擎

- simple_chat.py：State 充当上下文历史信息

- interrupt.py：「AI 自动处理 + 人类中断审核」的 LangGraph 工作流

- command.py：Command 基本使用

- checkpoint.py：存储快照
